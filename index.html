7. Application Layer

The application layer is used by end-user software such as web browsers and email clients. It provides protocols that allow users to send and receive information and present meaningful data. Examples of application layer protocols include Hypertext Transfer Protocol (HTTP), File Transfer Protocol (FTP), Post Office Protocol (POP), Simple Mail Transfer Protocol (SMTP), and Domain Name System (DNS).

6. Presentation Layer

The presentation layer prepares data for the application layer. It defines how devices encode, encrypt, and compress data so it is received correctly on the other end. It prepares data for transmission over the session layer.

5. Session Layer

The session layer creates communication channels called sessions between devices. It is responsible for opening sessions, keeping them active during data transfer, and closing them when communication ends.
